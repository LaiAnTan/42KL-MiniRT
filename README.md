# 42KL-MiniRT
A basic raytracing engine.

## Features

- CPU based rendering
- Objects (Plane, Sphere, Cylinder, Cone)
- Colored & multiple spot lighting
- Full Phong Reflection Model (Diffuse, Specular, Ambient)
- Texture support (for spheres)

## Dependencies

- gcc w/ Math Library
- MinilibX Library
- GNU Make
