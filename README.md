# 42KL-MiniRT
A simple raytracing engine.

## Features

- CPU based rendering
- Objects (Plane, Sphere, Cylinder, Cone)
- Colored & multiple spot lighting
- Full Phong Reflection Model (Diffuse, Specular, Ambient)
- Textures && Bump Maps (soon)

## Dependencies

- gcc w/ Math Library
- MinilibX Library
- GNU Make
